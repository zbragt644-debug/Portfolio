About Me

I’m a junior full-stack web developer with about a year of experience developing responsive and user-friendly web applications. Based in the Philippines, I work primarily with Laravel PHP, and I enjoy building clean interfaces using Tailwind CSS, Bootstrap, and JavaScript. I’m passionate about learning new technologies and improving my craft with each project I take on.

My journey in web development started from a simple curiosity about how websites are built, which eventually grew into an interest in creating practical and user-friendly web applications. I try to write clean and understandable code, and I’m always learning new tools and techniques to improve my work.

Beyond web development, I also have hands-on experience with computer hardware assembly, maintenance, and troubleshooting. I understand how software and hardware interact, allowing me to diagnose and resolve performance or compatibility issues effectively. I also possess junior-level knowledge in networking — including basic network setup, configuration, and understanding how data flows across connected systems.

During my college years, I also studied Arduino programming and Internet of Things (IoT), where I learned how to integrate software with physical devices to create interactive and automated systems. This background gives me a strong understanding of how web technologies, hardware, and networking come together in real-world applications.
